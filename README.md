AREISPRO - SISTEMA DE RELATÓRIOS
O AREISPRO nasceu da necessidade de otimizar e agilizar o processo de criação de relatórios fotográficos, transformando uma tarefa manual e repetitiva em um fluxo de trabalho automatizado e eficiente.
O Propósito: Automatização e Velocidade
O principal objetivo deste aplicativo é reduzir pela metade o tempo gasto na elaboração de relatórios técnicos e fotográficos. Ao automatizar a organização das imagens, a numeração das figuras e a formatação do documento final em PDF, o AREISPRO elimina os passos manuais que consomem tempo.
Uma Solução Independente e Inspirada
Este é um projeto desenvolvido por conta própria, fruto de iniciativa pessoal e inspirado nos desafios e rotinas do meu trabalho atual. A vivência prática permitiu identificar os principais gargalos no processo de relatórios e criar uma ferramenta sob medida para solucioná-los.
Principais Funcionalidades
Atualmente, o AREISPRO oferece uma interface intuitiva (estilo iOS) para:
•	Gestão de Dados: Cadastro e seleção rápida de Prestadores de Serviço e Lojas.
•	Relatório Fotográfico: Upload simplificado de fotos (Fachada e Marquise) e inserção de múltiplos Anexos (fotos Antes e Depois) com legendas e observações.
•	Geração de PDF: Compilação automática de todo o conteúdo em um documento PDF profissional, formatado, com páginas personalizadas.
Futuras Atualizações e Manutenção
Gostaríamos de informar que o módulo de Conversor de Arquivos (PDF para Word, etc.) ainda está em manutenção. Esta funcionalidade está sendo aprimorada e será disponibilizada em uma atualização futura, junto com outros serviços adicionais que estão em desenvolvimento para tornar o AREISPRO uma plataforma ainda mais completa.
1.	Tela de Boas-Vindas
Esta é a porta de entrada do aplicativo. O design foi pensado para ser minimalista e moderno, inspirado na estética (glassmorphism).
Objetivo: Criar uma experiência de usuário (UX) fluida e limpa desde o primeiro segundo.



 
1.	Painel Principal
Este painel é o coração da produtividade do AREISPRO. O propósito de sua criação foi eliminar o processo manual de montagem de relatórios.
•	Por que usar: Em vez de gastar horas copiando fotos, digitando legendas e ajustando a paginação no Word, aqui o usuário apenas insere os dados e as imagens em um fluxo guiado.
•	O Resultado: Isso reduz o tempo de elaboração do relatório técnico pela metade, garantindo padronização e profissionalismo no documento final em PDF, gerado instantaneamente.



















2.	Painel Principal
Botão Home: Garante a navegabilidade do usuário, permitindo o retorno imediato à tela principal de preenchimento de qualquer seção do aplicativo.

Coluna de Informações (Esquerda): Seleção de Prestador e Loja via banco de dados para evitar erros de digitação. Inclui campos para Nº de Chamado e Data.				
Fotos Fachada/Marquise: 
Espaço padronizado para Fachada e Marquise, garantindo o padrão visual exigido pela empresa em todos os relatórios.
Sistema de Anexos:
Fluxo inteligente onde o upload gera automaticamente uma página no PDF. Se o campo Observação Geral for preenchido, o sistema gera um Índice automático; se ficar vazio, a página de anexo é descartada.
Gestão de Imagens: Opção de excluir fotos enviadas incorretamente, adicionar descrições individuais e criar novos anexos conforme a necessidade do serviço.
Configurar Logos: O botão abaixo das informações permite trocar as logomarcas que aparecerão na capa do PDF (Cliente vs. Prestador).		
Botão Dark Mode (Power/Theme): Alterna instantaneamente entre os modos Claro e Escuro, adaptando a interface para o conforto visual em diferentes ambientes de trabalho.
Botão Gerar PDF: Gatilho principal que compila todos os dados, fotos e anexos em um documento profissional com paginação automática e capa personalizada.
3.	Painel Principal
Botão Conversor: Atalho para o hub de transformação de arquivos (ex: PDF para Word), atualmente em fase de manutenção para futuras implementações.
								







Menu de Configurações: Acesso rápido ao gerenciamento de Lojas e Fornecedores.
•	Botão Sair: Retorna o usuário à tela de boas-vindas ("Começar").
•	Segurança (Futuro): Estrutura pensada para a implementação futura de login e senha, garantindo acesso restrito e personalizado.




 
4.	Painel de Configurações: Autonomia e Cadastro

A tela de Configurações é pensada para dar total autonomia ao usuário, permitindo personalizar os dados que alimentam a tela principal sem a necessidade de alterar o código do aplicativo.
 
Como utilizar:
1.	Digite o nome da Loja ou Empresa no campo correspondente (o texto será convertido automaticamente para maiúsculas).
2.	Clique em "ADICIONAR" para salvar no banco de dados local.
3.	Use o botão "×" vermelho ao lado de cada item para removê-lo.
4.	Ao finalizar, clique em "Salvar e Voltar" para retornar ao Painel Principal com os dados atualizados nos menus de seleção.
